The 2nd is the Android 4.3 (API level 18) version which will run on Blackberry 10 devices (Passport, Classic, Q10, Z10, Z30, Q5, Z3 and Leap) and old Android phones.

Please download and install the below files in the given sequence, depending on your phone.


For Symbian S60 3rd Edition Feature Pack 1 & 2 phones (e.g. Nokia N95, E72, E90 etc.), download the below files. Please set your phone date to 2011 for installing Qt 4.6.3 and Qt Mobility 1.02 files as the certificate has expired. Then you can set it back to current date.
  1. Download Qt 4.6.3 runtime files
  2. Download Qt Mobility 1.02 runtime files
  3. Download Nokia WhatsApp v1.03 Qt app

For Symbian^1 phones, a.k.a. Symbian S60 5th Edition (e.g. Nokia N97, 5800 XpressMusic etc.), download the below files.
  1. Download Qt 4.7.3 runtime files
  2. Download Nokia WhatsApp v1.04 Qt app

For Symbian^3 phones, download the below Qt runtime files. Note that Symbian Anna and Symbian Belle phones already come pre-installed with Qt 4.7.3 or higher, so you don't need to install Qt runtime files.

  1. Download Qt 4.7.3 runtime files
  2. Download Nokia WhatsApp v1.04 Qt app


After installing the Qt runtime files and Nokia WhatsApp Qt app, click the below link to login once to create the WhatsApp session on my server.

This is a one time setup required to authenticate your login via Web and link your device. After this you can start using the app.

The login page uses Web Sockets so please use a modern browser like latest version of Chrome or Firefox.
